description: Learn what data the ShareX application and website use, how third-party upload services work, and which privacy policies apply.
---

The ShareX application does not collect data of any kind, including personally identifiable information.

ShareX sends files (such as images, text, and videos) selected or captured by the user only to third-party services that the user chooses and configures.

ShareX uses the GitHub API to check for updates at startup and periodically thereafter. ShareX downloads updates from GitHub only when the user permits it.

ShareX uses YouTube API Services to facilitate video uploads to YouTube. By using ShareX's API clients for YouTube services, users agree to comply with the YouTube Terms of Service.

Our website uses Google Analytics to measure page views.

ShareX encourages users to review the privacy policies of the third-party services with which it integrates:

* [GitHub Privacy Statement](https://docs.github.com/en/github/site-policy/github-privacy-statement)
* [Google Privacy & Terms](https://policies.google.com/privacy)
* [YouTube Terms of Service](https://www.youtube.com/t/terms/)
* [YouTube API Terms of Service](https://developers.google.com/youtube/terms/api-services-terms-of-service)
* [Google Analytics Terms of Service](https://marketingplatform.google.com/about/analytics/terms/us/)

Please review these documents to understand how your information is handled when using these third-party services.

Users can contact the development team through [GitHub Issues](https://github.com/ShareX/ShareX/issues).
