# XerahS Images

Place screenshots and images for the XerahS microsite here.

## Structure

- `placeholder.png`: Default placeholder image for missing screenshots.
- `xerahs/`: Folder for XerahS application screenshots.
- `editor/`: Folder for XerahS Editor screenshots.

## Usage

Reference images in HTML using `/xerahs/img/filename.png` or `/xerahs/img/folder/filename.png`.
